PyDevices / palettes

Color palettes for pydisplay

palettes is a small, portable color toolkit built for pydisplay demos and apps. It gives you wheels, cubes, Material-style swatches, and other curated collections so you can pick readable, on-theme colors without hard-coding hex values in every example.

Install with mip from the PyDevices micropython-lib index, or pull a CPython wheel from TestPyPI. The same import palettes path shows up in the browser gallery (via MIP) and on device firmware — keep it beside pydisplay whenever an example needs more than a handful of named colors.

Documentation GitHub Browser demos

Install

import mip
mip.install("palettes", index="https://PyDevices.github.io/micropython-lib/mip/PyDevices")
